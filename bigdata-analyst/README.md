## Skill 2: SQL数据分析

### 功能
复杂SQL查询与分析

### 核心能力
- 复杂多表关联
- 窗口函数
- 递归查询
- 性能优化

### 示例
```sql
-- 留存分析
SELECT 
    dt,
    COUNT(DISTINCT user_id) as new_users,
    COUNT(DISTINCT CASE WHEN days_diff = 1 THEN user_id END) as d1_retention,
    COUNT(DISTINCT CASE WHEN days_diff = 7 THEN user_id END) as d7_retention
FROM (
    SELECT 
        user_id,
        MIN(dt) over(PARTITION BY user_id) as first_dt,
        dt,
        DATEDIFF(dt, MIN(dt) over(PARTITION BY user_id)) as days_diff
    FROM user_behavior
) t
GROUP BY dt
```
